Description of the various test SQL files- zc154.zip. The result of a latin1 installation for zc154    - Note that        - The media_clips table has invalid 0000-00-00 dates.        - The database's overall collation is utf8, even though the installation was for a latin1 database.    - When creating the database, its overall collation should be latin1_general_ci.    - categories_descriptions table, description for "Printers" has some extended alphabetic characters for collation conversion testing.
